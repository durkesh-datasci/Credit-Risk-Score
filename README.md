# Credit Risk Scoring Model

This project is a simple statistical credit scoring engine that evaluates customer credit risk based on:

* Income
* EMI
* Credit Utilization
* Late Payments

## 📊 Sample Output

Customer C1 -> Score: 900
Customer C2 -> Score: 710
Customer C3 -> Score: 900
Customer C4 -> Score: 620
Customer C5 -> Score: 790

## ⚙️ How it Works

The model uses a rule-based scoring logic inspired by credit bureau systems (like CIBIL).

## ▶️ Run the Project

```bash
python src/main.py
```
